# IslandCollection

搜索A岛， K岛等系列岛上一定人气的串的图形化软件。

目前支持:

- A-主岛(h.nimingban.com)  
- A-养老岛(h.adnmb.com)
- A-光驱岛（新旧都可以）

计划中:
k岛系列， 2cat系列。  

计划功能:
- proxy

特性
-------
- 本地收藏夹  
- 支持对结果进行过滤， 搜索  
- 对多个岛的支持  
- 自动过滤文本中多余的html标签
